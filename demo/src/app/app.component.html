Smart Form Error Mapper

Zero-boilerplate, centralized validation messages

① Directive: appError

Add appError to any input — error span injected automatically on blur.

② Component: <form-error>

Explicit placement — useful when input and error live in different DOM nodes.

③ Instant feedback: [errorOnDirty]="true"

Errors appear as you type — no need to blur first.
